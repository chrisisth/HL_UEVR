# HL_UEVR
A VR first person mod for Hogwarts Legacy using UEVR
With added script to block noMovement calls and stop vertical movement while casting.

## Features
- **First-person VR experience** with full 6DOF head tracking
- **Motion-controlled wand** attached to VR controllers or hand models
- **Gesture-based spell casting** - draw spell glyphs in 3D space
- **Movement lock prevention** - blocks unwanted "noMoving" context for smooth VR gameplay
- **Root motion elimination** - prevents unwanted forward steps during spell casting and combat animations
- **Multiple locomotion modes** - head-directed, hand-directed, or traditional game controls
- **VR hand models** with dynamic animations
- **Mount support** - optimized camera positions for brooms, Hippogriffs, and other creatures
- **Adaptive UI** - menu system with VR-friendly positioning options
